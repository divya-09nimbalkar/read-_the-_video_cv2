# read-_the-_video_cv2


**OpenCV Webcam Video Processing (Python)**

This project demonstrates basic real-time video processing using the OpenCV (cv2) library in Python. The program accesses the system webcam, captures live video frames, converts each frame into grayscale format, and displays the processed video in a window.

It illustrates how OpenCV can be used for camera access, frame capture, image processing, and real-time visualization. These concepts form the foundation for many computer vision applications such as face detection, surveillance systems, motion detection, and AI-based vision systems.


**Key Concepts Used:**

Webcam access using cv2.VideoCapture()

Capturing frames from a video stream

Converting frames to grayscale using cv2.cvtColor()

Displaying video frames using cv2.imshow()

Terminating the program using a keyboard key (q)



**How the Program Works:**

The program connects to the system webcam.

It continuously reads frames from the camera.

Each frame is converted from BGR color format to grayscale.

The processed frame is displayed in real time in a window.

Press q to exit the program and close the video window.
